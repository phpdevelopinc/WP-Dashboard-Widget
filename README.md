
# WP Dashboard Widget

## Node Version

`V16`

## Install

`npm install`

## Usage

`npm start`

## Rest API Endpoint

`/wp-json/chardata/v1/data?filter=<value>`

 - <value> : 0 - Display All data
 - <value> : 7 - Display 7 Days Data
 - <value> : 15 - Display 15 Days Data
 - <value> : 1 - Display 1 month Data
